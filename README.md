# Festival Sales Analysis

## Overview

This project focuses on analyzing festival season sales data to uncover customer purchasing behavior, product performance, and sales trends. Using Python and data analytics techniques, the project provides actionable insights that can help businesses improve customer targeting, inventory planning, and marketing strategies during festive seasons.

---

## Objectives

* Analyze customer purchasing patterns during festival seasons.
* Identify top-performing product categories.
* Understand customer demographics and buying behavior.
* Generate business insights to improve sales performance.
* Visualize key trends using charts and graphs.

---

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## Dataset

The dataset contains festival sales transaction records, including:

* Customer Demographics
* Gender
* Age Group
* State
* Occupation
* Product Category
* Purchase Amount

The data was analyzed to identify trends and customer preferences during the festival season.
